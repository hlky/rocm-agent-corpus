# gfx12 RDNA4 Consumer Alias

This directory is kept as a compatibility pointer for older corpus links. Use
`architecture/gfx1200_rdna4/README.md` for current RDNA4 guidance.

Do not add measurements directly under this alias. Records should cite the exact
reported `gfx` target, usually `gfx1200` or another target reported by
`rocminfo` and supported by `hipcc --list-gpu-targets`.
