# Examples

These examples are reference sketches for agents. They are not all wired into
CI yet. Treat them as starting points that should be compiled, tested, and
curated before becoming measured corpus tasks.

## Directories

- `examples/hipblaslt`: descriptor-driven matmul and epilogue sketches.
- `examples/composable-kernel`: Composable Kernel/CK Tile customization sketches.
- `examples/rocprim`: hipCUB/rocPRIM/hipCUB/rocThrust primitive sketches.
- `examples/rocthrust`: rocThrust algorithm composition sketches.
